# Java 11 ★★★★★

Java 11 is the first Long Term Support (LTS) release after Java 8 and is widely used in enterprise applications.

---

# HttpClient API ★★★★★

## Interview Answer

Java 11 introduced a modern HttpClient API to replace HttpURLConnection.

Supports

- HTTP/1.1
- HTTP/2
- Synchronous calls
- Asynchronous calls

Example

```java
HttpClient client = HttpClient.newHttpClient();
```

Advantages

- Cleaner API
- Async support
- Better performance

# String API Enhancements ★★★★☆

## isBlank()

```java
"   ".isBlank();
```

Returns true for empty or whitespace strings.

---

## lines()

```java
text.lines();
```

Splits string into Stream<String>.

---

## repeat()

```java
"Hi".repeat(3);
```

Output

```
HiHiHi
```

---

## strip()

Removes Unicode whitespace.

Preferred over trim().

# var (Java 10) ★★★★★

## Interview Answer

var allows local variable type inference.

Compiler determines the type.

Example

```java
var list = new ArrayList<String>();
```

Compiler treats it as

```java
ArrayList<String>
```

Restrictions

- Only local variables
- Cannot be method parameter
- Cannot be return type
- Cannot be field

# Records ★★★★★

## Interview Answer

Records provide a concise way to create immutable data classes.

Instead of writing

- constructor
- getters
- equals()
- hashCode()
- toString()

Java generates them automatically.

Example

```java
record Employee(int id,
                String name) {}
```

Equivalent to an immutable POJO.

Use Cases

- DTOs
- API Responses
- Immutable Models

---

# Sealed Classes ★★★★☆

## Interview Answer

Sealed classes restrict which classes can extend them.

Example

```java
public sealed class Shape
permits Circle, Rectangle {

}
```

Advantages

- Better control
- Improved maintainability
- Safer inheritance

# Pattern Matching for instanceof ★★★★☆

Before Java 17

```java
if(obj instanceof String){

    String s = (String)obj;
}
```

Java 17

```java
if(obj instanceof String s){

}
```

No explicit casting required.

Cleaner and safer.

# Switch Expressions ★★★★☆

Switch can return values.

Example

```java
String day =
switch(num){

    case 1 -> "Monday";

    case 2 -> "Tuesday";

    default -> "Unknown";
};
```

Advantages

- Less boilerplate
- No fall-through
- Cleaner syntax

# Virtual Threads ★★★★★

## Interview Answer

Virtual Threads are lightweight threads managed by the JVM instead of the operating system.

Create using

```java
Thread.startVirtualThread(() -> {

});
```

Advantages

- Millions of threads
- Low memory usage
- Better scalability
- Ideal for I/O intensive applications

Use Cases

- REST APIs
- Database calls
- Network applications

---

# Virtual Threads vs Platform Threads ★★★★★

| Platform Thread | Virtual Thread |
|-----------------|----------------|
| OS Managed | JVM Managed |
| Heavyweight | Lightweight |
| Expensive | Cheap |
| Limited | Millions possible |
